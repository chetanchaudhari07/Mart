**Mart Backend API**

A Node.js + Express REST API for managing Users, Products, and Transactions with a structured architecture using Controllers → Services → Repositories and a Supabase PostgreSQL database.

This project also demonstrates optimized data handling and scalable backend structure suitable for dashboards and data analytics applications.

**Features**

User management API, Product management API, Transaction tracking, Supabase PostgreSQL integration, Clean architecture pattern, Optimized database queries, Indexing for performance, RESTful API design


**Tech Stack**

*Backend*

Node.js, Express.js

*Database* 

Supabase (PostgreSQL)



Installation

*Clone the repository:* 
`` git clone https://github.com/chetanchaudhari07/Mart.git ``

*Go to backend folder:* 
``cd Mart/Backend_Mart``

*Install dependencies:*

``npm install``

*Run development server:*

``npm run dev``

*Run production server:*

``npm start``

*Environment Variables*

Create a .env file in the root folder:

``SUPABASE_URL=your_supabase_project_url``
``SUPABASE_KEY=your_supabase_anon_key``
``PORT=3000``
